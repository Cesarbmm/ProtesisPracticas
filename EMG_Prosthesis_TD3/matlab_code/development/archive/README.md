# Development Archive Inventory

Inventario estable del archivo historico de `development/`.

## Carpetas

- `check_dataset_legacy/`: inspecciones tempranas del dataset anterior a la linea actual
- `parallel_computing_trials/`: pruebas antiguas de paralelizacion no integradas al flujo vigente
- `adapt_dataset_denis_raw/`: scripts y datos fuente de conversion del dataset Denis
- `simulator_creation_assets/`: scripts y artefactos usados para construir el simulador y sus `.mat`

## Criterio

- migrar solo lo que vuelva a ser necesario para runtime o documentacion tecnica
- preservar sin mezclar el resto como procedencia reproducible
