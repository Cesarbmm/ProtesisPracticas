# Analysis

Esta carpeta queda reservada para analisis mantenido y portable.

## Estado actual

- `legacy/`: viewers historicos con rutas absolutas antiguas
- `legacy_config_copy/`: copia historica del duplicado que antes estaba en `config/analysis/`

## Regla

No colocar aqui viewers nuevos que dependan de rutas locales fijas.
Si se recupera alguno, debe rehacerse para rutas relativas y entonces volver a la raiz de `analysis/`.
